PAPER DETAILS SUMMARY
=====================

Proposed title
--------------
Diabetes Status Classification with Calibrated XGBoost and Conformal Prediction: External Transportability and Reliability Assessment

Study purpose
-------------
This study develops a reproducible machine-learning workflow for diabetes-status classification using demographic and clinical variables. The primary goals are to: (1) compare standard machine-learning classifiers; (2) select and calibrate XGBoost; (3) quantify uncertainty using standard split conformal prediction and class-conditional (Mondrian) conformal prediction; (4) assess clinically relevant sensitivity-specificity trade-offs; and (5) report uncertainty, subgroup performance, and reproducibility artefacts transparently.

The study should be presented as diabetes-status classification using the variables available in the public dataset. It should not be presented as a prospective early-diagnosis system, an autonomous clinical tool, or a ready-to-deploy diagnostic device. HbA1c and blood glucose are closely related to clinical diabetes assessment and contribute strongly to the observed predictive performance.

Preprocessing
-------------
Numeric predictors: median imputation.
Categorical predictors: most-frequent imputation and one-hot encoding.
Logistic regression and SVM: numeric standardization in the training pipeline.
Decision tree, random forest, and XGBoost: no numeric standardization.
All transformations are fitted within model pipelines to prevent information leakage.

Proposed model
--------------
Final point-prediction model: probability-calibrated XGBoost classifier.

Base algorithm:
  - XGBoost binary logistic classifier
  - Objective: binary:logistic
  - Evaluation metric during fitting: log loss

Model selection:
  - Five-fold stratified cross-validation on the training partition only
  - Selection criterion: ROC-AUC

Selected XGBoost hyperparameters:
  - n_estimators: 600
  - max_depth: 3
  - learning_rate: 0.05
  - subsample: 1.0
  - colsample_bytree: 0.8
  - min_child_weight: 1
  - reg_lambda: 5.0
  - Best cross-validated ROC-AUC: 0.9799

Probability calibration:
  - Sigmoid calibration
  - Five-fold cross-validation within the training partition
  - The held-out calibration partition is NOT used to refit the point-prediction model.

Uncertainty model:
  1. Standard split conformal prediction.
  2. Class-conditional (Mondrian) split conformal prediction.

For calibration observation (x_i, y_i), the nonconformity score is:
  s_i = 1 - p_model(y_i | x_i)

Standard split conformal uses one finite-sample threshold for both classes.
Mondrian conformal uses independent finite-sample thresholds for class 0 and class 1.

For a test record, a class is included in the conformal set when:
  1 - p_model(class | x) <= conformal threshold

Coverage definition:
  Coverage is the proportion of test records for which the observed label is included in the prediction set. Coverage is NOT the proportion of non-empty sets.

Recommended manuscript message
------------------------------
The paper's central result is not simply that XGBoost achieved high ROC-AUC. The defensible contribution is a reproducible evaluation showing that:
  - point-prediction performance changes materially with the operating threshold;
  - standard conformal prediction can satisfy marginal coverage while under-covering the diabetes class;
  - Mondrian conformal prediction recovers near-nominal class-wise coverage while exposing uncertainty through prediction-set size;
  - all estimates are accompanied by fixed splits, bootstrap confidence intervals, calibration evaluation, and descriptive subgroup analyses.

Limitations that must remain in the paper
------------------------------------------
1. NHANES evaluates a five-feature harmonized reduced model, not the full eight-feature primary model.
2. The NHANES outcome is self-reported clinician-diagnosed diabetes and differs from the source outcome definition.
3. HbA1c and blood glucose are closely related to diabetes assessment; the model is not evidence of pre-diagnostic prediction.
4. Conformal validity assumes exchangeability between calibration and future observations; NHANES empirically shows that source conformal thresholds did not fully transport.
5. Mondrian conformal provides class-conditional, not demographic-group-conditional, coverage.
6. Subgroup analyses are descriptive and may have small event counts in some strata.
7. The Pima analysis is a small secondary benchmark, not direct external validation.
8. The balanced sensitivity analysis does not represent the observed disease prevalence.
